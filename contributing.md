We welcome contributions that improve the mathematical visualization, code structure, documentation, or educational value of hi-pie!

## How to Contribute

1. **Fork the repository**
2. **Clone your fork**
3. **Create a new branch**
4. **Make your changes** (code, docs, etc.)
5. **Test locally** to ensure your contribution works
6. **Push to your fork**
7. **Open a Pull Request** targeting the `main` branch

## What Can You Work On?

- Improve performance or animation quality
- Add new mathematical patterns or formulas
- Enhance documentation or explanations
- Add example images/gifs to the README
- Improve code comments for clarity
- Test compatibility on different Python versions/platforms

## Code Style

- Use Python 3.x conventions
- Keep code readable and well-commented
- Use `black` for auto-formatting if possible
- Write concise commit messages

## Reporting Issues

If you spot a bug or request a feature, open an [Issue](https://github.com/willow788/hi-pie/issues) with details, screenshots, or instructions to reproduce if applicable.

## Questions & Discussions

Curious about the mathematics? Want to suggest a new type of visualization?  
Open a [Discussion](https://github.com/willow788/hi-pie/discussions) or connect with the author via GitHub.
